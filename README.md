# modern-frontend-framework
comparson similarity,contrast difference 学习next.js,react,vue之间的相同与不同
## React 认为“UI 是函数的返回值”，倾向用 JS 表达一切（包括结构） ui界面即函数返回值
![alt text](图片.png)
## Vue 认为“模板更直观”，用声明式模板 + 响应式数据驱动更新   ref是响应式的 可以跟踪变量的变化 template模板支持{{}}插入值 @click事件等
## Nextjs 约定优于配置的路由系统,页面组件都要默认放到src/app目录下 组件可以在服务端渲染 可以在组件中直接获取数据
